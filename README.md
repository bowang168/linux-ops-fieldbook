# Linux Ops Fieldbook

**Production Troubleshooting & Performance Guide for RHEL-based Linux**

> *"In production, you don't need to know everything — you need to know where to look in 5 minutes."*

A practical, structured reference for experienced Linux engineers, SREs, DevOps practitioners, and technical support professionals. All commands and configurations are based on **RHEL 8/9 and compatible distributions** (Rocky, Alma, Oracle Linux).
